# OpenBoardnet - manufacturing-data
## Introduction
The manufacturing-data repository contains the necessary files for producing the wire harnesses based on the OpenBoardnet car developed in the KI4Boardnet project.
As input we used the .kbl files provided by EDAG Engineering GmbH.

## Files
### Formboards
Manufacturing Drawings (Formboards) from the tool EPLAN harness proD 2025 in PDF and DXF formats:
- Fender_Front_Right.pdf
- Fender_Front_Right.dxf
- Chassis_Front_Right.pdf
- Chassis_Front_Right.dxf
- Bumper_Front_Right.pdf
- Bumper_Front_Right.dxf
- Engine_Compartment_Front_Right.pdf
- Engine_Compartment_Front_Right.dxf

### Bill of material
Bill of material for manufacturing the harnesses in XLSX format:
- Fender_Front_Right_BOM.xlsx
- Chassis_Front_Right_BOM.xslx
- Bumper_Front_Right_BOM.xslx
- Engine_Compartment_Front_Right_BOM.xslx
